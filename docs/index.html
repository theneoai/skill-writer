核心特性

行业领先的 AI 技能质量保障体系

🎯

1000分评估体系

4阶段深度评测：结构验证 → 文本质量 → 运行时测试 → 综合认证，量化每一个技能的质量。

  • Phase 1: 结构完整性 (100分)
  • Phase 2: 文本质量分析 (300分)
  • Phase 3: 运行时测试 (400分)
  • Phase 4: 综合认证 (200分)
🤖

多LLM审议机制

借鉴司法陪审团制度，3个LLM独立评分后交叉验证，消除单一模型的偏见和幻觉。

  • Generator: 生成初稿
  • Reviewer: 安全审计
  • Arbiter: 仲裁分歧
  • 共识矩阵确保公正
🔄

自进化系统

3触发器自动监控技能质量，发现问题自动优化，无需人工干预。

  • Threshold: 质量阈值监控
  • Time: 新鲜度检查
  • Usage: 使用率分析
  • 7维度9步优化循环
🌏

原生双语支持

中英文无缝切换，所有模板内置双语触发词，评估报告双语输出。

  • 双语触发词识别
  • 混合语言输入支持
  • 双语评估报告
  • 国际化模板

API Tester

API Integration

HTTP API 测试自动化，支持 TEST/VALIDATE/BATCH 三种模式，自动环境变量注入，完整的错误处理。

# 测试单个接口
skill run api-tester \
  --input '{"url": "https://api.example.com/users"}'

# 批量测试
skill run api-tester --mode batch \
  --input '{"endpoints": [...]}'

快速开始

5分钟上手 Skill Framework

1

克隆仓库

git clone https://github.com/theneoai/skill.git
cd skill
2

创建技能

cp templates/base.md my-skill.md
# 编辑 my-skill.md，填写你的技能定义
3

快速评估

# 运行 LEAN 评估（约1秒）
./skill-framework.sh lean my-skill.md
4

完整评测

# 运行完整评估（4阶段1000分）
./skill-framework.sh evaluate my-skill.md
