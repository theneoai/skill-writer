核心特性

行业领先的 AI 技能质量保障体系

⚡

LEAN 快评模式

500分启发式快评，约1秒完成，无需调用 LLM。8项检查快速判断技能质量。

  • YAML frontmatter 完整性
  • §N 格式段落检查
  • Red Lines / 严禁文本
  • 触发词覆盖度
🎯

1000分评估体系

4阶段深度评测：结构验证 → 文本质量 → 运行时测试 → 综合认证，量化每一个技能的质量。

  • Phase 1: 结构完整性 (100分)
  • Phase 2: 文本质量分析 (300分)
  • Phase 3: 运行时测试 (400分)
  • Phase 4: 综合认证 (200分)
🤖

多LLM审议机制

借鉴司法陪审团制度，3个LLM独立评分后交叉验证，消除单一模型的偏见和幻觉。

  • Generator: 生成初稿
  • Reviewer: 安全审计
  • Arbiter: 仲裁分歧
  • 共识矩阵确保公正
🔄

自进化系统 (UTE)

Use-to-Evolve 协议，3触发器自动监控技能质量，发现问题自动优化，无需人工干预。

  • Threshold: 质量阈值监控
  • Time: 新鲜度检查
  • Usage: 使用率分析
  • 7维度9步优化循环

API Tester

API Integration

HTTP API 测试自动化，支持 TEST/VALIDATE/BATCH 三种模式，自动环境变量注入，完整的错误处理。

# 测试单个接口
skill run api-tester \
  --input '{"url": "https://api.example.com/users"}'

# 批量测试
skill run api-tester --mode batch \
  --input '{"endpoints": [...]}'
