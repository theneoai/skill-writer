一键安装

选择平台，复制命令，粘贴到你的 AI 助手即可

curl -fsSL https://raw.githubusercontent.com/theneoai/skill-writer/main/install.sh | bash

安装到本机所有支持的 AI 平台（Claude / OpenCode / OpenClaw / Cursor / Gemini / OpenAI / MCP）

read https://github.com/theneoai/skill-writer/releases/latest/download/skill-writer-claude.md and install to claude

安装到 ~/.claude/skills/，同时写入 refs / templates / eval / optimize 伴侣文件

read https://github.com/theneoai/skill-writer/releases/latest/download/skill-writer-opencode.md and install to opencode

安装到 ~/.config/opencode/skills/

read https://github.com/theneoai/skill-writer/releases/latest/download/skill-writer-openclaw.md and install to openclaw

安装到 ~/.openclaw/skills/

read https://github.com/theneoai/skill-writer/releases/latest/download/skill-writer-cursor.md and install to cursor

安装到 ~/.cursor/skills/

read https://github.com/theneoai/skill-writer/releases/latest/download/skill-writer-gemini.md and install to gemini

安装到 ~/.gemini/skills/

read https://github.com/theneoai/skill-writer/releases/latest/download/skill-writer-mcp.json and install to mcp

安装 MCP JSON manifest 到 ~/.mcp/servers/skill-writer/mcp-manifest.json

核心特性

行业领先的 AI 技能质量保障体系

⚡

LEAN 快评模式

17项结构检查，[STATIC] 确定性检查（335分，零方差） + [HEURISTIC] LLM启发式检查（165分），快速判断技能质量。

  • YAML frontmatter 完整性 [STATIC]
  • §N 格式段落 + Quality Gates [STATIC]
  • 触发词 EN/ZH 覆盖度 [STATIC]
  • 红线/错误处理/安全基线 [STATIC]
  • 模板匹配度 + 字段具体性 [HEURISTIC]
🎯

1000分评估体系

4阶段深度评测：结构验证 → 文本质量 → 运行时测试 → 综合认证，量化每一个技能的质量。

  • Phase 1: 结构完整性 (100分)
  • Phase 2: 文本质量分析 (300分)
  • Phase 3: 运行时测试 (400分)
  • Phase 4: 综合认证 (200分)
🤖

多遍自审协议

Generate/Review/Reconcile 三遍自审，在单一 LLM 会话内用独立视角发现盲点，VERIFY 步骤消除分数膨胀。

  • Generate: 生成初稿技能
  • Review: 安全审计 + 质量检查
  • Reconcile: 差异整合 + 修正
  • VERIFY: 收敛后独立再评估
🔄

自进化系统 (UTE)

Use-to-Evolve 协议，3触发器自动监控技能质量，发现问题自动优化，无需人工干预。

  • Threshold: 质量阈值监控
  • Time: 新鲜度检查
  • Usage: 使用率分析
  • 7维度9步优化循环
